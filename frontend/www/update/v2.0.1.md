# v2.0.1 업데이트 내역

이번 업데이트는 새로운 콘텐츠 추가, 게임 밸런스 조정, 그리고 관리자 기능 강화에 중점을 두었습니다.

## ✨ 주요 개선 사항

### 1. 신규 콘텐츠 추가
- **무한 모드 (300층 이상)**: 300층을 돌파하면 '무한 모드'가 시작됩니다. 기존의 모든 몬스터가 능력치가 대폭 강화되어 끝없이 등장하며, 50층마다 몬스터 수가 1마리씩 늘어납니다. 진정한 한계에 도전해보세요!
- **신규 장비 추가**: 최상위 등급의 무기 5종과 방어구 5종이 새롭게 추가되었습니다. 더 깊은 동굴을 탐험하고 강력한 장비를 획득하세요.

### 2. 게임 밸런스 조정
- **능력치 최대치 설정**: 게임의 밸런스를 위해 일부 능력치에 최대 한도가 적용되었습니다.
    - **치명타 확률**: 최대 50%
    - **흑섬 확률**: 최대 50%
    - **회피 확률**: 최대 60% (기존과 동일)
- **스탯 UI 개선**: 스탯 분배 창에서 각 능력치의 최대치가 표시되며, 최대치에 도달하면 수치가 강조되어 표시됩니다.

### 3. 편의성 및 UI/UX 개선
- **모바일 UI 최적화**:
    - 몬스터가 3마리 이상 등장할 경우, 좌우로 스크롤하여 모든 몬스터를 확인할 수 있도록 개선했습니다.
    - 작은 화면에서도 버튼이 잘리지 않도록 컨트롤 버튼과 장비 모달 내 버튼들의 크기를 동적으로 조정했습니다.
