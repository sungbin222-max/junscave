# v2.1.0 업데이트 내역

이번 업데이트는 모바일 플레이 경험을 대폭 개선하고, 상점 UI를 재구성하며, 다양한 편의 기능을 추가했습니다.

## ✨ 주요 개선 사항

### 1. 모바일 UX 대폭 개선
- **화면 꺼짐 방지**: 게임 플레이 중 화면이 자동으로 꺼지지 않도록 Wake Lock 기능이 추가되었습니다.
- **햅틱 피드백**: 전투 중 흑섬, 치명타, 레벨업 등 주요 이벤트 발생 시 진동으로 알려드립니다.
- **스와이프 제스처**: 모달 창을 아래로 스와이프하여 닫을 수 있으며, 전투 화면에서 좌우 스와이프로 대상 몬스터를 변경할 수 있습니다.
- **가로 모드 지원**: 가로 화면에서 정보 패널과 전투 영역이 2열 레이아웃으로 표시됩니다.

### 2. 상점 UI 개편
- **탭 기반 네비게이션**: 상점이 물약/장비/내 장비 3개의 탭으로 정리되어, 원하는 항목을 빠르게 찾을 수 있습니다.
- **모바일 스크롤 개선**: 작은 화면에서 모달 내부 스크롤이 더 매끄럽게 동작합니다.

### 3. 성능 최적화
- **플로팅 텍스트 최적화**: 데미지/힐 텍스트 표시에 오브젝트 풀 패턴을 적용하여, 자동 전투 시 성능이 크게 향상되었습니다.

### 4. 알림 기능
- **게임 오버 알림**: 브라우저 탭이 비활성 상태일 때 캐릭터가 사망하면 알림으로 알려드립니다.

### 5. 버그 수정
- **서버 연결 오류 수정**: 리버스 프록시 환경에서 API 경로가 올바르게 설정되지 않던 문제를 수정했습니다.
